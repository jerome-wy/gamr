## **Description:**
gamr is a Full Stack Web Application that uses Python with the Django Framework for the Back-End and uses React and React Libraries for the Front-End. 

## **Technologies Used:**

**Back-End**

- Python
- Django
- PostgreSQL
- Node.js
- Sequelize
- Bcrypt
- Jsonwebtoken

**Front-End:**

- HTML5
- CSS3
- JavaScript ES6
- React.js
- Bootstrap
- Tailwind

**React Libraries:**
- React-Icons       
- React-Recharts
- React-Toastify
- Semantic UI

**Applications:**
- Adobe Photoshop
- Whimsical
- Lucid Chart
